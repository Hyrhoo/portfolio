Description du Projet

Ce projet consistait à imaginer et concevoir une base de données pour une entreprise vendant des produits. La base de données devait gérer les produits, les commandes, les producteurs, les clients, les adresses internationales et les taxes.

Ce projet a été réalisé en groupe de quatre et nous avons choisi de modéliser une entreprise type Ikea vendant des kits de meubles. En tant que passionné de conception de bases de données, j'ai pris l'initiative de la conception et je sollicitais les avis de mes coéquipiers.

De par le domaine de l'entreprise choisi, nous avons dû créé des tables pour les meubles et les pièces les composant, ainsi que pour les fabricants de ces pièces. Une fois la base de données créée, nous avons effectué des requêtes. J'ai été le principale acteur de cette tâche en raison de mes compétences en bases de données. La complexité de notre base de données a rendu cette étape plus difficile que prévu.
